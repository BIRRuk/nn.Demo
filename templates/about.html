Usage and Troubleshooting

Upload good quality images of chest radiographs that contain all the whole lung field.

The images should be of greater than 224 pixels in height and width. images of lesser dimenssion return a small image size error

Use an aspect ratio close to a square, images that are too wide or tall return a bad aspect ratio error

Use images of ['png', 'jpg', 'jpeg', 'gif', 'dcm'] format, other formats return a unsupported file format error
